pinyin
======

convert Simplified Chinese to pinyin in golang

install
--------

	 go get github.com/axgle/pinyin

example
-------

	package main
	import (
	 "github.com/axgle/pinyin"
	)
	func main() {	 
	  println(pinyin.Convert("hello,世界!"))
	}
